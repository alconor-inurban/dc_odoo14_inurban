CARGAR DE REGISTROS A MODELO CON CARACTERES ESPECIALES COMO: "Ñ", "Á, É . . . .

1. CONSTRUYA EL ARCHIVO CARGADOR EN EXCEL CON EL ENCABEZADO CON LOS NOMBRE DE CAMPOS DEL MODELO DESEADO
2. IMPORTE MANUALMENTE EN LA VISTA LISTA DEL MODELO DESEADO
3. CONECTE A SU BASE DE DATOS POSTGRESQL CON DBEAVER
4. BUSQUE LA TABLA DEL MODLEO EN CUESTION Y ABRA LA OPCION: VIEW DATA Y VAYA A LA OPCION: DATA PARA VERLA
5. USA LA OPCION EXPORT RESULT PARA EL DATA TRANSFERT
6. USE LA OPCION CSV
7. ASEGURES DE QUE LA OPCION ENCODING ESTE EN "UTF-8" Y LA OPCION DE CHECK LIST DE COPY TO CLIPBOARD
8. COPIE DEL CLIPBOARD A SU ARCHIVO EL CONTENIDO.

ESTO GARANTIZA QUE LOS CARACTES ESPECIALES SE TRADUZCAN EN CODIGO UTF-8
